Fridge Allocation Project

This project is about a system that allows consumers to bring in scrap fridges in for repairs and consumers who want to purchase second hand fridges to be able to get a fridge at an affordable price.

This web app is built through an open source framework, namely Visual Studio ASP.NET MVC. This web app performs CRUD operations, such as the user logging in to order a scrap fridge or leave a scrap fridge, creates an account if they don’t already have one, updates information and deletes invalid information on their profile.

We also used Microsoft SQL server to manage our web app through stored procedures that helped to guide our code operation through any query and ensure data integrity.


Technologies Used

  • HTML, CSS, JavaScript
  • ASP.Net MVC
  • Microsoft SQL Server
